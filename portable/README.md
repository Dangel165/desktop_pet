# 데스크탑 펫

PyQt5로 만든 간단한 데스크탑 펫입니다.

화면 위를 돌아다니고, 마우스를 따라오거나 자유롭게 산책할 수 있습니다. 오른쪽 클릭 메뉴에서 대화할 수 있고, OpenAI API 키를 넣으면 AI 채팅도 사용할 수 있습니다.

## 어떻게 만들었는지

이 프로그램은 투명한 PyQt5 창을 화면 위에 띄우고, 그 안에 캐릭터 이미지를 보여주는 방식으로 만들었습니다.

창은 테두리가 없고 항상 위에 뜨도록 설정했습니다. 그래서 일반 프로그램 창처럼 보이지 않고, 바탕화면 위를 돌아다니는 캐릭터처럼 보입니다.

펫의 움직임은 타이머로 계속 위치를 조금씩 바꾸는 방식입니다. 마우스 따라가기 모드에서는 현재 마우스 위치를 향해 속도를 부드럽게 바꾸며 이동합니다. 자유 산책 모드에서는 랜덤 방향과 속도를 조금씩 바꿔서 자연스럽게 돌아다니게 했습니다.

대화 기능은 작은 채팅창을 따로 띄우는 방식입니다. 기본 상태에서는 코드 안에 있는 간단한 로컬 답변을 사용하고, OpenAI API 키를 입력하면 OpenAI Responses API로 답변을 받아옵니다.

## 기술 스택

- Python: 전체 프로그램 작성
- PyQt5: 투명 창, 메뉴, 대화창, 타이머, 이미지 표시
- QMainWindow: 바탕화면 위에 떠 있는 펫 창
- QLabel: 캐릭터 이미지 또는 GIF 표시
- QMovie: GIF 애니메이션 재생
- QTimer: 이동, 투명도 변화, 애니메이션 갱신
- QMenu: 오른쪽 클릭 메뉴
- QInputDialog: 이름 바꾸기 입력창
- QDialog: OpenAI 키 설정창
- QThread: OpenAI 응답을 기다리는 동안 화면이 멈추지 않게 처리
- urllib: OpenAI API 요청 전송
- json: 설정 파일 저장과 API 데이터 처리

## 주요 기능 구조

- `PetBrain`: 펫의 대화 담당
- `ChatWindow`: 펫과 대화하는 채팅창
- `OpenAISettingsDialog`: OpenAI API 키와 모델을 설정하는 창
- `OpenAIReplyWorker`: AI 답변을 백그라운드에서 받아오는 작업
- `Sticker`: 실제로 화면 위를 돌아다니는 펫

## 구현 포인트

- 투명 배경을 사용해서 캐릭터만 떠 있는 것처럼 보이게 만들었습니다.
- `WindowStaysOnTopHint`로 다른 창 위에 보이게 했습니다.
- `FramelessWindowHint`로 창 테두리와 제목 표시줄을 없앴습니다.
- 마우스 위치는 `QCursor.pos()`로 읽습니다.
- 마우스 따라가기는 바로 순간 이동하지 않고 속도를 보간해서 부드럽게 따라오게 했습니다.
- 마우스 따라가기 중에는 커서와의 거리, 커서 속도에 따라 표정과 행동이 바뀝니다.
- 자유 산책은 순간이동하지 않고 속도를 조금씩 바꾸는 방식으로 자연스럽게 만들었습니다.
- 기본 캐릭터는 코드에서 직접 그리며, 자유 산책 중 랜덤 표정과 행동을 합니다.
- 외부 `character.gif`나 `character.png`를 넣으면 원하는 캐릭터로 바꿀 수 있습니다.
- OpenAI API 키는 `pet_config.json`에 저장됩니다.

## 실행 방법

아래 파일을 더블클릭하면 됩니다.

```powershell
run_desktop_pet.bat
```

배치파일은 `py.exe`, `python.exe`, 자주 쓰이는 Python 설치 경로를 자동으로 찾아서 실행합니다. 그래서 환경변수 설정이 없어도 실행할 수 있습니다.

직접 실행하고 싶으면 다음 명령을 사용하세요.

```powershell
py desktop_pet.py
```

## Python이 없는 컴퓨터에서 실행하기

다른 컴퓨터에 Python이나 PyQt5가 없어도 실행할 수 있도록 휴대용 EXE를 만들 수 있습니다.

먼저 개발 컴퓨터에서 아래 파일을 더블클릭합니다.

```powershell
build_portable.bat
```

빌드가 끝나면 `portable` 폴더가 생깁니다. 이 폴더 전체를 다른 Windows 컴퓨터로 옮긴 뒤 아래 파일을 더블클릭하면 됩니다.

```powershell
run_portable.bat
```

다른 컴퓨터에는 Python, PyQt5, 환경변수 설정이 필요하지 않습니다.

원하는 캐릭터 이미지도 함께 사용하려면 `portable` 폴더 안에 `character.gif` 또는 `character.png`를 넣으세요.

## 중복 실행 방지

이미 데스크탑 펫이 실행 중이면 두 번째 펫은 실행되지 않습니다.

중복 실행을 시도하면 "데스크탑 펫이 이미 실행 중입니다."라는 안내창이 뜨고 종료됩니다.

## 실행할 때 캐릭터 선택하기

프로그램을 실행할 때마다 캐릭터 선택 창이 뜹니다.

- `기존 캐릭터`: 마지막으로 선택한 캐릭터를 사용합니다. 선택한 캐릭터가 없으면 기본 캐릭터를 사용합니다.
- `새로운 캐릭터`: 움직이는 프레임 폴더 또는 PNG/GIF 파일 하나를 선택합니다.

새 캐릭터 파일은 아래 폴더에 넣으면 됩니다.

```text
characters
```

움직이는 캐릭터는 PNG 프레임이 들어 있는 폴더를 통째로 선택하면 됩니다.

단일 캐릭터는 PNG/GIF/JPG/BMP 파일 하나를 선택하면 됩니다.

다른 위치의 폴더나 파일을 선택하면 프로그램이 자동으로 `characters` 폴더 안으로 복사한 뒤 사용합니다.

새 캐릭터를 선택하면 크기 배율도 같이 정할 수 있습니다.

- `100%`: 원래 크기
- `50%`: 절반 크기
- `200%`: 두 배 크기

가로/세로 비율은 유지되므로 캐릭터가 납작하게 찌그러지지 않습니다.

## 조작 방법

- 마우스 이동: 펫이 커서를 따라옵니다.
- `Ctrl` 누르기: 펫이 선명하게 보입니다.
- `Ctrl` 떼기: 펫이 다시 반투명해집니다.
- 펫 왼쪽 클릭: 아무 일도 일어나지 않습니다.
- 펫 오른쪽 클릭: 대화와 설정 메뉴를 엽니다.

## 오른쪽 클릭 메뉴

- `대화하기`: 펫과 대화합니다.
- `이름 바꾸기`: 펫 이름을 새로 정합니다.
- `OpenAI 키 설정`: OpenAI API 키와 모델을 설정합니다.
- `AI 채팅 켜기/끄기`: 로컬 대화와 AI 대화를 바꿉니다.
- `자유 산책하기`: 마우스를 따라오지 않고 자유롭게 돌아다닙니다.
- `마우스 따라가기`: 다시 마우스를 따라옵니다.
- `종료`: 프로그램을 종료합니다.

대화 중에는 펫이 잠깐 멈춥니다. 대화창을 닫으면 원래 모드로 돌아갑니다.

## 자유 산책

자유 산책 모드에서는 펫이 화면 안에서 자연스럽게 돌아다닙니다.

기본 캐릭터를 사용할 때는 랜덤으로 표정과 행동도 바뀝니다.

- 눈 깜빡임
- 웃는 표정
- 놀란 표정
- 옆 보는 표정
- 졸린 표정
- 통통 튀는 움직임

## 마우스 상호작용

마우스 따라가기 모드에서는 기본 캐릭터가 커서에 반응합니다.

- 커서가 아주 가까움: 좋아하는 표정
- 커서가 빠르게 움직임: 놀란 표정
- 커서가 적당히 가까움: 바라보는 표정
- 커서가 멀어짐: 놀라서 따라감
- 가끔 눈 깜빡임

## OpenAI AI 채팅

1. 펫을 오른쪽 클릭합니다.
2. `OpenAI 키 설정`을 누릅니다.
3. API 키를 입력합니다.
4. `AI 채팅 사용`을 체크합니다.
5. 저장합니다.

이후 대화창에서 말을 걸면 OpenAI API로 답변합니다.

API 키와 설정은 이 폴더의 `pet_config.json`에 저장됩니다. 키를 지우고 싶으면 `pet_config.json` 파일을 삭제하면 됩니다.

## 원하는 캐릭터 사용하기

단일 캐릭터는 `characters` 폴더에 아래 파일 중 하나를 넣으면 사용할 수 있습니다.

- `character.gif`
- `character.png`
- `character.jpg`
- `character.bmp`

움직이는 캐릭터를 프레임별로 쓰고 싶다면 캐릭터 전용 폴더를 만들고 이런 이름으로 넣을 수 있습니다.

- `idle_right_0.png`
- `idle_right_1.png`
- `walk_right_0.png`
- `walk_right_1.png`
- `walk_left_0.png`
- `walk_left_1.png`

예시:

```text
characters/my_pet/idle_right_0.png
characters/my_pet/idle_right_1.png
characters/my_pet/walk_right_0.png
characters/my_pet/walk_right_1.png
characters/my_pet/walk_left_0.png
characters/my_pet/walk_left_1.png
```

프로그램 실행 시 `새로운 캐릭터` → `프레임 폴더`를 선택하고 `characters/my_pet` 폴더를 고르면 됩니다.

이렇게 넣으면 이동 방향과 상태에 따라 PNG 프레임 애니메이션이 재생됩니다.

프레임 폴더 캐릭터도 크기 배율을 적용할 수 있으며, 모든 프레임은 같은 비율로 확대/축소됩니다.

## 필요한 라이브러리

```powershell
py -m pip install PyQt5
```

이미 PyQt5가 설치되어 있다면 따로 설치할 필요가 없습니다.
